# Components

This section documents the React components used in the Example Frontend application.

## Component Overview

The application consists of several key components organized into logical groups:

### Core Components

- [Header](header.md) - Main navigation header
- [App](app.md) - Root application component

### Page Components

- [Contract Management](pages/contract-management.md) - Contract management interface
- [Jury Selection](pages/jury-selection.md) - Jury selection process
- [Query Definition](pages/query-definition.md) - Query definition interface
- [Results](pages/results.md) - Results display
- [Run Query](pages/run-query.md) - Query execution interface

### Utility Components

- [Paginated Justification](utility/paginated-justification.md) - Pagination component

## Component Architecture

Components follow React best practices with clear separation of concerns and reusable design patterns.
